Rasa shi-tzu

Shih tzu este un caine de talie mica, robust, dar cu o inima mare. Cu un nume care inseamna leu in limba chineza, Shih tzu este o rasa de caini foarte veche si un animal de companie foarte popular. Recunoscuta ca rasa membra a grupei cainilor de agrement de catre American Kennel Club in 1969, Shih tzu este o excelenta alegere pentru familiile cu copii sau pentru persoanele in etate.

Aspectul exterior si dimensiunile

Shih tzu este un caine robust, asemanator rasei Lhasa Apso, acoperit in intregime de o blana dubla formata din fire de par lungi si un puf profund pufos. Capul acestor caini este rotund, cu un bot acoperit cu peri scurti, cu barba si mustata dese. Stopul este evident. Ochii sunt mari, rotunzi, caprui sau negrii. Coada stufoasa este rasucita peste spate. Shih tzu are un nas scurt si turtit, de culoare neagra sau maro, la cainii cu roba rosu-vinetiu.

Personalitatea

Shih tzu a fost creat pentru a fi un animal de companie si nimic altceva. Rasa este rezistenta si robusta, cea mai putin delicata dintre rasele ''de jucarie". Dau impresia ca sunt mandri si aroganti, dar, in realitate sunt tandri si foarte jucausi. Shih tzu este un caine mititel ager si plin de viata. Este un caine de paza vigilent, caruia ii place sa latre, dar in general este tacut in interiorul casei. Shih tzu este inteligent, dar si incapatanat.

Shih tzu are tendinta de a fornai si de a sforai si nu tolereaza bine caldurile excesive.
